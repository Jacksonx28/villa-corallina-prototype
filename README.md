# Villa Corallina - Modern Luxury Prototype

This is a high-fidelity landing page prototype for Villa Corallina, Moorea. It features a cinematic video header, responsive design for luxury branding, and a functional inquiry form.

## How to Play With This Project
If you would like to tweak the design or test new features:

1. Fork this repository - (Click the 'Fork' button at the top right).
2. Clone your fork - to your local machine or edit directly on GitHub.
3. Enable GitHub Pages - in your fork's 'Settings' tab to see your changes live.

## Tech Stack
- HTML5 & CSS3 (Custom Grid/Flexbox)
- Google Fonts (Cormorant Garamond & Montserrat)
- Mobile-responsive video background
